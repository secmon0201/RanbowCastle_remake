# 🏰 Rainbow Castle Remake (彩虹城堡重制版)

> ⚠️ **Work In Progress** | 🚧 开发中

这是一个基于 **RPG Maker MZ** 引擎开发的《彩虹城堡》重制版项目。

## 📖 项目简介 (Introduction)

* **引擎**：RPG Maker MZ
* **类型**：回合制 RPG
* **核心目标**：在还原经典剧情的基础上，提升游戏体验，保留经典元素，优化RM原生内容。

## ✨ 特色系统 (Features)

本项目包含大量独立开发的 JavaScript 插件系统 ，用于定制战斗系统、UI 界面、角色系统等。

### ⚔️ 深度定制战斗系统
* **UI 重构**：重写战斗界面，脱离RM原版风格。
* **战斗表现**：
    * 引入 Moghunter 的动态立绘。
    * 自定义敌人血条。
    * 更加流畅的战斗动画处理。
    * 战斗时间轴系统，提供更具策略性的行动顺序。
    * 状态窗口固定与优化 。

### 🧬 角色养成与系统
* **天赋系统**：独立开发的天赋加点菜单。
* **数值成长**：自定义职业成长曲线。
* **新手引导**：内置教程与引导系统。

## 开发说明

### 资源管理
- 所有游戏资源均存放在相应的目录中
- 音频文件支持 OGG 和 M4A 格式
- 图像文件主要为 PNG 格式

### 游戏数据
- 游戏数据采用 JSON 格式存储
- 可以使用 RPG Maker MZ 编辑器或文本编辑器修改
- 地图数据包含地图布局、事件和触发器

## 游戏内容

### 角色系统
- 多个可玩角色，每个角色都有独特的技能和属性
- 丰富的装备系统，包括武器、防具和饰品
- 角色升级和技能学习系统

### 战斗系统
- 回合制战斗
- 多种元素属性（物理、火焰、雷电等）
- 丰富的技能和魔法

### 地图与场景
- 超过 50 个游戏地图
- 多样化的场景设计（城镇、森林、洞穴等）
- 隐藏的宝藏和秘密区域
